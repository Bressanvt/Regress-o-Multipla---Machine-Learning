# Regressão Múltipla

Regressão linear múltipla implementada do zero com NumPy. Prediz preços de casas usando múltiplos atributos do dataset kc_house_data (quartos, banheiros, andares, etc).

**Conceitos:** múltiplas variáveis independentes, equação com vários coeficientes, diferença da regressão linear simples, comparação de abordagens (menor diferença vs padrão).
